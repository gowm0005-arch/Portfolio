# M. Gowtham — Portfolio

A premium, responsive personal portfolio built with React + Vite, Tailwind CSS v4, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Customize

- Edit content (projects, skills, contact info, etc.) in `src/data/content.js`.
- Replace the GitHub username placeholder in `src/components/GitHubSection.jsx` with your real GitHub handle.
- Add your resume PDF to `public/resume.pdf` so the "Download Resume" button works.
- Replace the "MG" monogram in `Hero.jsx` / `About.jsx` with a real photo by dropping an image into `src/assets` and swapping the markup.
- Update real GitHub/LinkedIn links and project links in `src/data/content.js`.

## Tech Stack

React 19, Vite, Tailwind CSS v4, Framer Motion, React Icons.

## Features

Dark/light mode toggle, typewriter hero text, glassmorphism cards, animated skill bars, scroll-reveal animations, scroll progress bar, scroll-to-top button, cursor glow (desktop), page loader, animated counters, fully responsive layout, accessible focus states, and reduced-motion support.
